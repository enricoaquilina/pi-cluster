# and openclaw-heavy/slave0/slave1/router-api showing down in MC.
#
# These deps live in their own venv (see scripts/install-openclaw-router-venv.sh)
# and are consumed by the openclaw-router-api.service systemd unit.
#
# If you bump fastapi, also update the starlette pin — FastAPI's upper
# bound moves with every minor release. Keep starlette inside FastAPI's
# allowed range (check `pip show fastapi` → Requires-Dist).
